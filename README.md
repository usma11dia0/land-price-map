# 地価情報マップ

地価情報を地図上で確認できるWebアプリケーション

## 概要

不動産鑑定士向けの地価情報可視化ツールです。公示地価・基準地価などの地価情報を地図上にマッピングし、直感的に確認・分析できます。

## 主な機能（予定）

- 地図上での地価ポイント表示
- 公示地価・基準地価の検索
- 地価の推移グラフ表示
- エリア別の地価比較
- 地価データのフィルタリング（用途地域、価格帯など）

## 技術スタック（予定）

- **フロントエンド**: React / TypeScript
- **地図ライブラリ**: Leaflet または Mapbox
- **バックエンド**: Node.js / Express（必要に応じて）
- **データソース**: 国土交通省 地価公示データ

## データソース

- [国土交通省 地価公示](https://www.mlit.go.jp/totikensangyo/totikensangyo_fr4_000043.html)
- [国土交通省 都道府県地価調査](https://www.mlit.go.jp/totikensangyo/totikensangyo_fr4_000044.html)

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```
